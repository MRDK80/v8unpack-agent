# elem.json и form_elements_index (elem_parser)

После распаковки обычной формы агент видит не только `Form.obj.bsl`, но и
`*.elem.json`. `Form.obj.bsl` содержит обработчики, а `elem.json` — структуру
формы: группы, панели, кнопки, поля, привязки и страницы.

```python
from pathlib import Path
from v8unpack_agent import parse_elem_json

result = parse_elem_json(Path("unpacked/Form/ФормаЭлемента"))

if result.elem_index_ok:
    print(result.elements)
else:
    print(result.warnings)
```

Если `elem_index_ok=False`, основной текстовый слой формы остаётся доступен.
Это не ошибка распаковки, а сигнал о неполноте структурного контекста.

## Нормализованное дерево для elem_sha256

Для вычисления `elem_sha256` (issue #40) хэшируется **нормализованное** дерево
элементов — только структурно значимые поля:

| Поле | Назначение |
|---|---|
| `name` | Имя элемента |
| `type` | Тип элемента (`InputField`, `Button`, …) |
| `path` | Путь в дереве |
| `parent` / `parent_path` | Родительский элемент |
| `page` | Страница формы |
| `source` / `data_path` | Привязка реквизита |
| `handler` | Обработчик события |

Косметические поля (`left`, `top`, `width`, `height`, `color`, `font`, GUID и
пр.) **исключены** — сдвиг элемента на пиксель или смена цвета не дают
`structure_modified`.

## Граница достоверности

Вложенность **групп** не реконструируется из публичных артефактов — она только
в raw/info-векторах (недокументированный формат платформы). `parse_elem_json`
честно помечает это в `warnings`. Хэш строится по тому, что достоверно:
панели, страницы, плоский список элементов страницы, привязки, обработчики.

## forms_index и forms_scan_index

`forms_index` (FormsIndex) — **карта актуальности** для pipeline-шага
распаковки. Содержит:

| Поле | Тип | Значение |
|---|---|---|
| `bin_path` | string | Путь к исходному `Form.bin` |
| `unpacked_root` | string | Директория с распакованными текстами |
| `bin_mtime` | float | Unix-время изменения `Form.bin` при распаковке |
| `unpacked_mtime` | float | Unix-время последней распаковки; `bin_mtime > unpacked_mtime` → форма устарела |
| `extraction_ok` | bool | `true` — полная распаковка |
| `warnings` | array | Диагностика при частичной распаковке |

Этот индекс не является источником истины и не содержит содержимого `Form.bin`,
строк подключения или имён баз — его безопасно коммитить в репозиторий.
